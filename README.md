# Lost Deal Recovery Radar (MVP)

Tutorial-style lead generation/retention tool for title/referral businesses.

## Value proposition
Leadership can instantly see:
- which referral partners are likely churning,
- how much quarterly revenue is at risk,
- and who to contact first to win back deals.

## Run locally
```bash
npm install
npm start
```
Open http://localhost:3000

## CSV format
`partner_name,deals_last_90d,deals_prior_90d,avg_deal_value,days_since_last_deal,response_rate,partner_tenure_months`

## Demo flow
1. Click **Load Demo Dataset**
2. Walk through **Leadership Walkthrough** steps
3. Show at-risk revenue and top win-back list
